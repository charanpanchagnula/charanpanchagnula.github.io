# Featured Projects

## Security Remediation Intelligence System
*Prototype | Multi-Agent AI Platform*

An **agentic AI platform** that automatically identifies and fixes security vulnerabilities. Unlike traditional scanners that generate noise, this system uses a multi-agent architecture to analyze findings, generate context-aware code patches, and validate them for safety and correctness.

![Security Remediation System Architecture](/security-remediation-arch.png)

### Key Capabilities

- **Reflection Pattern (Generator-Evaluator):** Implements an agentic workflow design pattern where the Generator Agent produces fixes and the Evaluator Agent validates them in a feedback loop
- **Multi-Scanner Aggregation:** Unified view for SAST (Semgrep), IaC (Checkov), and SCA (Trivy)
- **S3 Vectors for RAG:** Uses AWS S3-based vector storage for semantic search on historical remediations
- **Infrastructure as Code:** Includes Terraform code for deployment to AWS App Runner
- **False Positive Detection:** Automated triage with confidence scoring

**Technologies:** Python, FastAPI, Next.js, Agno, LangChain, RAG, AWS (App Runner, SQS, S3, DynamoDB), Terraform

[View on GitHub →](https://github.com/charanpanchagnula/security-remediation-system)

## MCP Scanner Benchmark & Evaluation System

A comprehensive framework for **benchmarking and evaluating security scanners** for the Model Context Protocol (MCP). Uses an Agentic AI approach to analyze scan results, score tools based on performance, and maintain a holistic leaderboard.

![MCP Scanner Benchmark Architecture](/mcp-scanner-arch.png)

### Key Features

- **Multi-Scanner Orchestration:** Runs multiple open-source MCP security scanners in parallel
- **AI-Driven Evaluation:** Uses DeepSeek (via Agno) to review findings, assign percentage scores, and determine confidence levels
- **Holistic Leaderboard:** Tracks long-term scanner performance using a moving average system
- **Dual-Mode Analysis:** Supports both Static Analysis and Dynamic Fuzzing

**Technologies:** Python, FastAPI, Next.js, Agno, DeepSeek, Docker, MCP Protocol

[View on GitHub →](https://github.com/charanpanchagnula/mcp-scanner-benchmark)

## Internal Security Operations Chatbot

### Tool-Based Scanner Operations
- **DynamoDB Integration:** Tool calls to query scanner execution data, track failures, identify stuck jobs, and enable one-click resubmission of security scans
